# BibTeX files for publications

Add one `.bib` file per publication. In `src/content/publications.yaml`, set each publication’s **`bibtex`** field to the path under `public/`, e.g.:

- `"bibtex": "bibtex/1.bib"` → file `public/bibtex/1.bib`
- `"bibtex": "bibtex/paper-name.bib"` → file `public/bibtex/paper-name.bib`

Omit `bibtex` for publications that don’t need a Cite button.
